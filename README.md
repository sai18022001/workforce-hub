\# Workforce Hub



A full-stack Employee \& Project Management System built with:



\- \*\*Frontend:\*\* React.js + TypeScript + Vite + TailwindCSS + Apollo Client

\- \*\*Backend:\*\* Go + gqlgen (GraphQL) + EntGo (ORM)

\- \*\*Database:\*\* MySQL

\- \*\*DevOps:\*\* Docker + Docker Compose + GitHub Actions CI/CD



\## Features

\- JWT-based authentication with role management (Admin, Manager, Employee)

\- Employee, Department, Project, Assignment, and Timesheet management

\- GraphQL API with queries, mutations, and subscriptions

\- Real-time dashboard with charts and analytics

\- Fully containerized and CI/CD automated
